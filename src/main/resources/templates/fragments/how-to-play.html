How to play

  1. You see a code snippet where each word is hidden (shown as tildes). Only the language is given up front.
  2. Type a word you think appears in the code. If it is there, every occurrence is revealed. Wrong guesses are listed; repeating the same word does not cost extra tries.
  3. Peek next word reveals the next hidden token in reading order and counts as one hint.
  4. After enough of the code is uncovered (about 35% decoded) or you have used two hints, you can submit a final answer: the program, class, or function name the snippet belongs to.
  5. Show answer ends the round, shows the solution and full code, and scores zero for that attempt.
